# All-of-the-mod-9-hybird
Modified the modpack all of the mod 9 for make it compatible with Arclight.

# French Section
J'ai modifier le modpack All of the mod 9 pour faire en sorte qu'il sois compatible avec le serveur Hybride Arclight.

## Comment j'ai fais ?
Tous d'abord, pour ouvrir le modpack, j'ai utiliser le launcher Prism qui peut ouvrir les modspack fais sous Forge (en plus, quand on crée une instance, on peut aussi le télécharger dans Prsim.).

Je voulais de base que les mods serveur, je suis donc aller sur la page Forge du mod, puis aller voir dans les fichiers facultatif. Ensuite, j'ai pu le mettre sur le serveur.

Sauf que j'ai un serveur Arclight et certains mod, ce n'est pas possible.

Pour faire le tri entre les mods valide et non valide, je l'es ai ajouter par ordre alphabétique, groupe par groupe (Exemple: Mettre les A d'abord puis ensuite les B), et je lancé le serveur avec ces mod là.

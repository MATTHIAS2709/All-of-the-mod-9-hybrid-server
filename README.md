# All-of-the-mod-9-hybrid
Modified the modpack all of the mod 9 for make it compatible with Arclight.

# French Section
J'ai modifier le modpack All of the mod 9 pour faire en sorte qu'il sois compatible avec le serveur Hybride Arclight.

## Comment j'ai fais ?
Tous d'abord, pour ouvrir le modpack, j'ai utiliser le launcher Prism qui peut ouvrir les modspack fais sous Forge (en plus, quand on crée une instance, on peut aussi le télécharger dans Prsim.).

Je voulais de base que les mods serveur, je suis donc aller sur la page Forge du mod, puis aller voir dans les fichiers facultatif. Ensuite, j'ai pu le mettre sur le serveur.

Sauf que j'ai un serveur Arclight et certains mod, ce n'est pas possible.

Pour faire le tri entre les mods valide et non valide, je l'es ai ajouter par ordre alphabétique, groupe par groupe (Exemple: Mettre les A d'abord puis ensuite les B), et je lancé le serveur avec ces mod là. J'ai fais ça tout en ajoutant avec les dépendances qu'ils avaient (Généralemnt, des library. Donc rare qu'il sois le problème).

# English Section
## How did I do it?
First of all, to open the modpack, I used the Prism launcher, which can open modpacks made with Forge (plus, when you create an instance, you can also download it in Prism).

I basically wanted the server mods, so I went to the mod's Forge page and looked in the optional files. Then I was able to put it on the server.

Except that I have an Arclight server and some mods aren't possible.

To sort through the valid and invalid mods, I added them in alphabetical order, group by group (e.g. put the A's first, then the B's), and launched the server with those mods. I did this while adding the dependencies they had (usually libraries, so it's rare that this is the problem).
